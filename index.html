What NEOKey does

A secure vault for secrets and codes with fast everyday UX — optimized for offline use and quick copy/reveal actions.

Encrypted vault

All sensitive data is encrypted at rest so secrets remain protected even when stored on-device.

Biometrics + PIN fallback

Unlock quickly with fingerprint/face, with a fallback PIN/password when biometrics fail.

CRUD entries

Create, read, update, and delete encrypted entries with reveal/hide and copy actions.

Categories & search

Organize entries into categories, filter them, and search to find what you need instantly.

Password generator

Generate strong passwords with adjustable options, a strength meter, and one-tap copy/insert.

Offline-first by design

No internet dependency — better privacy, fewer points of failure, and reliable access anywhere.

Concept & Tech Stack

Built with modern Android tools for a clean UI and secure local storage.

App concept
Provide a fast, simple, and secure way to store and use authentication codes and passwords without requiring internet access, while keeping sensitive data encrypted and protected by biometric unlock and a master pincode.
  • Store multiple entries (services, accounts, secrets, passwords)
  • Reveal/hide codes quickly and copy when needed
  • Generate strong passwords inside the app
  • Organize with categories (and optionally share securely later)
Tech stack
Kotlin Jetpack Compose Room (SQLite) Offline-first architecture Encryption layer Android Keystore
Core features (from spec)
  • Local SQLite database (Room)
  • Reveal/hide stored codes
  • Password strength meter
  • Biometric unlock
  • Encryption & decryption
